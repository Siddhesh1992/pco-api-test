Technologies & Tools

Requirements

For running the application,you will only Node.js(v14 and above) and a package manager like npm installed in your environment.The NestJs CLI will also be needed to be installed globally. You will also need a database explorer like HeidiSQL, pgAdmin etc for connecting to the database.

Installation

  • Clone the pco-api repository from GitHub.
  • Then enter the pco-api directory using command cd pco-api.
  • Run the initial installation of all modules using command npm install.

External Integrations

Okta

  • Integration with Okta for token verification on login and fetching of user accounts details.

New Relic

  • Integration with New Relic to help application owners track the memory utilization of their services.

Service Now

  • Integration with Service Now to help application owners create requests.

Usage

Development Environment

  • Run the command npm run start:dev to start up the api server.

Production Environment

  • Run the command npm run build to create a build.
  • Then the command npm start to start the server.

The server should start running on port 4000.

Accessing the APIs

APIs details can be found in the requests.http file which can be used to send requests in VS Code using the REST Client extension or a third party software like Postman.

The list of APIs can also be found at the http://localhost:4000/api/ route integrated with the Swagger UI.

Testing

  • Test Suites can be run using the command npm run test.
